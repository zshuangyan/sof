Installation
------------

## From Source at Github

- Clone the repository

```sh
$ git clone https://github.com/zshuangyan/sof.git && cd sof
```

- Create virtualenv
```sh
$ python3 -m venv venv
$ source venv/bin/activate
```

- Install sof with setup.py

```sh
$ python setup.py install
```

## Usage
